# Download Progress Bar

<h3>## 🔗 Live Demo:- </h3>

A simple and interactive Download Progress Bar built using JavaScript, `setTimeout()` and `setInterval()`.

## 🚀 Features

- Simulates a file download process.
- Dynamic progress bar animation.
- Real-time percentage updates.
- Clean and responsive UI.
- Beginner-friendly JavaScript project.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 Concepts Practiced

- DOM Manipulation
- setTimeout()
- setInterval()
- Updating UI Dynamically
- Event Handling
- Progress Tracking Logic

## 🎯 How It Works

1. The download process starts when the user initiates the action.
2. JavaScript updates the progress value at regular intervals using `setInterval()`.
3. The progress bar width and percentage text are updated dynamically.
4. When the progress reaches 100%, the download is marked as complete.
